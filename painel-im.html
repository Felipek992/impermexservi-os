☁️ Sincronização via GitHub Gist
Configure uma vez e as alterações salvas aparecerão em qualquer dispositivo automaticamente — sem precisar exportar arquivos ou subir manualmente para o servidor.
▶ Acesse gist.github.com para criar seu Gist →
📋 Como configurar (uma vez só):
  1. Acesse gist.github.com, crie um novo Gist com o arquivo impermex-cfg.json e conteúdo {}. Marque como Secret e clique em Create.
  2. Copie o ID do Gist (os caracteres na URL após seu usuário).
  3. Vá em github.com/settings/tokens → Tokens (classic) → Generate new token. Marque apenas gist. Copie o token.
  4. Cole o Token e o ID abaixo e clique em 💾 Salvar Configuração.
  5. Pronto! O botão 💾 Salvar Tudo agora envia para o Gist e qualquer dispositivo carrega automaticamente.
Permissão necessária: apenas gist. Token fica salvo só neste navegador (localStorage).
URL do Gist: gist.github.com/seu-usuario/ID_AQUI
📤 Forçar envio ao Gist

Envia a configuração atual (salva no localStorage) para o Gist imediatamente. Útil para sincronizar após importar um backup.
